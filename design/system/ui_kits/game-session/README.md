# Game Session UI Kit

Recreation of Atlas's core gameplay HUD: map viewport, top resource/turn bar, minimap, unit inspector panel, action bar, and notification toasts.

No real game/map art was provided — the map viewport is an abstract terrain-tile placeholder (CSS grid of muted olive/umber tones), not real terrain art. Replace with real map rendering when available.

Icons: [Lucide](https://lucide.dev) via CDN (`unpkg.com/lucide`), per the iconography substitution noted in the root readme.
